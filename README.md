# The Odin Project | Calculator

A calculator and unit converter.


## Calculator
- Basic operations: addition, subtraciton, multiplication and division
- Indices and root functions
- Plus-minus function
- Factorial function
- Percentage function
- Able to take decimals as an input
- Able to take brackets as an input
- MRC, M+ and M- tools
- Backspace button

## Unit Converter
- Length
- Mass
- Volume
- Speed
- Time
- Energy
- Tempurature
- Currency
- Colour

## Additional Features
- Keyboard support
- Colour themes
